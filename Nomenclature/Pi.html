    Product Naming Rules¶

    CSAABBBRACCCD-EFGH
    CSAABBBRA4CCCD-EFGH

    CS

    Chipsee Product Abbreviation

    AA

    Horizontal Resolution:

    80 - 800px
    10 - 1024px
    12 - 1280px
    14 - 1440px
    19 - 1920px

    BBB

    Vertical Resolution:

    480 - 480px
    600 - 600px
    768 - 768px
    800 - 800px
    900 - 900px
    102 - 1024px
    108 - 1080px

    RA / RA4

    Based on Raspberry Pi Compute Module:

    RA - Raspberry Pi CM3
    RA4 - Raspberry Pi CM4

    CCC

    LCD Dimension:

    050 - 5.0”
    070 - 7.0”
    080 - 8.0”
    097 - 9.7”
    101 - 10.1”
    104 - 10.4”
    120 - 12.0”
    150 - 15.0”
    170 - 17.0”
    190 - 19.0”
    215 - 21.5”

    D

    Industrial Pi Type:

    E - Embedded PC without Case
    P - Panel PC with Case
    A - All-In-One Computer with Plastic Case

    E

    Touch Panel Type:

    R - Resistive Touch
    C - Capacitive Touch
    N - None

    F

    LCD Brightness:

    1 - Common Brightness
    2 - High Brightness

    G

    Baseboard PCB Version Number

    H

    PCB Version:

    SOM Module PCB Version Number
    CM4 Version Number
