.. |npu| replace:: Neural network acceleration engine with 6Tops int8, support Int4/8/16/FP16/BF16/TF32.

.. |PCIe| replace:: 1 x PCIe3.0 x4, M.2 M-Key 2230/2242/2260/2280 socket **(optional)** 

.. |pcie_opt_feature| replace:: The product has |PCIe|, you can use it with a NVMe SSD or other modules such as a AI compute module that can fit in the slot and supports the protocol. By default, SSD or AI module is **not included**; by default M.2 slot is **not mounted**. Please **contact us** before placing an order if you need M.2 slot or M.2 devices.

.. |uart_des| replace:: This product has 6 x UART channels in total. The default configuration is 2 x RS232 and 2 x RS485, including 1 debug port. CAN0 and CAN1 can be configured to RS232.

.. |tscreen| replace:: 10-point capacitive touch screen

.. |HDMI_des| replace:: 1 x HDMI 2.0 Out

.. |USB| replace:: 2 x USB 3.0 HOST, 1 x USB Type-C

.. |LAN| replace:: 1 x RJ45, GbE

.. |gpio_conn| replace:: **phoenix connector**

.. |GPIO| replace:: 8 Channels Isolated IO, 4 x Input and 4 x Output

.. |opt1| replace:: Supported, Optional

.. |opt2| replace:: Integrated WiFi/BT Module

.. |vin_rng| replace:: From 15V to 30V (supports **optional** 24V ignition signal)

.. |cur| replace:: 700mA max at 15V, 400mA typical at 15V

.. |pwr| replace:: 10.5W max, 6W typical

.. |temp| replace:: From -20°C to +70°C

.. |rs232| replace:: default 2 x RS232 (Optional 6 x RS232 at most, including 1 debug port)

.. |rs485| replace:: default 2 x RS485 at most

.. |can| replace:: default 2 x CAN FD

.. |poe| replace:: N/A

.. |which_usb_usbc_conflict| replace:: near HDMI

.. |usb_usbc_conflict| replace:: The USB-A host (near HDMI port) and USB-C **can't be used** at the same time. Before boot into OS, USB-C is enabled for install OS image; after boot into OS, USB-A is enabled but USB-C is disabled. In Android, these can be configured; in Linux, these can't be configured.

.. |LAN_des| replace:: 1 x RJ45

.. |vesa_screws| replace:: **75 x 75** mm or **100 x 100** mm, 4 x **M4** (6mm)
