Portable Battery Powered Tablet TV
##################################

.. include:: /PCs/Shared/four_main_images

Overview
========

.. rst-class:: text-center

    |product_img1|

This product is a portable TV with built in large capacity battery, it has 21.5 inch 10-point capacitive touch screen, running at 1920 x 1080 60Hz.

The product supports 2.4G and 5G WiFi, Bluetooth. It runs Android 12 with an 8 core CPU, it has 4GB RAM and 64GB internal storage.
